# To-Do List Web App (Flask + SQLite)

A simple web-based To-Do List application built with Flask.

## Features
- Add tasks
- View all tasks
- Delete tasks

## Tech Stack
- Python (Flask)
- SQLite
- HTML, CSS

## Run Locally
1. Install dependencies: `pip install flask`
2. Run app: `python app.py`
3. Open: `http://127.0.0.1:5000/`
